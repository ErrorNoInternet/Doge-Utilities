Privacy Policy

Bot Intents

Message Intent: Doge Utilities has the Message Intent, which means that it has access to the content in the messages you send. This intent is required for auto-moderation features, such as chat filters. There are 4 filters that use this intent: links, insults, mention, and newline. If a Discord server that you are in is using these features, please be informed that your messages will be logged (if they trigger the filters).
